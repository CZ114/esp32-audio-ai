# ESP32 Audio AI - 项目规格说明

## 1. 项目概述

- **项目名称**: ESP32 Audio AI
- **项目类型**: IoT语音采集 + AI分析系统
- **核心功能**: ESP32通过MEMS麦克风采集音频，通过WiFi传输到PC，PC端进行AI语音分析
- **目标用户**: 语音识别研究者、音频工程师、IoT爱好者

## 2. 硬件规格

### 2.1 硬件组件

| 组件 | 型号 | 数量 | 说明 |
|------|------|------|------|
| 主控 | ESP32-S3 | 1 | 双核240MHz, WiFi+蓝牙 |
| 麦克风 | INMP441 | 1 | I2S数字输出 MEMS |
| 电源 | USB-C 5V/2A | 1 | 通过ESP32 USB供电 |

### 2.2 引脚配置

| ESP32 GPIO | INMP441 | 功能 |
|-----------|---------|------|
| GPIO0 | SCK | I2S时钟 |
| GPIO1 | WS | 字选择 |
| GPIO2 | SD | 串行数据 |
| 3V3 | VDD | 3.3V电源 |
| GND | GND | 地 |

### 2.3 电源需求

| 组件 | 电压 | 电流 |
|------|------|------|
| ESP32S3 | 3.3V | 500mA (峰值) |
| INMP441 | 3.3V | 1.2mA |
| 总计 | 5V USB | ~600mA |

### 3.2 数据格式

#### UDP数据包
- 格式: 原始PCM二进制
- 采样率: 16000Hz
- 位深: 16bit
- 通道: 单声道
- 帧大小: 512样本 (32ms)

#### WAV文件
- 格式: RIFF WAVE
- 采样率: 16000Hz
- 位深: 16bit
- 通道: 1 (单声道)

## 4. 功能列表

### 4.1 ESP32固件 [esp32/]
- [x] WiFi连接 (Station模式)
- [x] I2S麦克风配置 (16kHz, 16bit, 单声道)
- [x] 串口命令控制 (start/stop)
- [x] UDP广播音频数据

### 4.2 PC客户端 [pc/]
- [x] UDP接收音频数据
- [x] 保存为WAV文件
- [x] 命令行参数支持 (端口、时长、输出文件)

### 4.3 AI分析 [ai_speech_analysis/]
- [x] Whisper语音转文字 (本地/API)
- [x] WebRTC停顿检测 (VAD)
- [x] jieba关键词提取
- [x] 语速计算 (字/分钟)
- [x] JSON结果输出

## 5. 通信协议

### 5.1 WiFi配置
- 模式: STA (Station)
- 传输: UDP广播
- 端口: 8888

### 5.2 串口命令
| 命令 | 功能 |
|------|------|
| start | 开始音频传输 |
| stop | 停止音频传输 |
